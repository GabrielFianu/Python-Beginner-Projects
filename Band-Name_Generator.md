# BAND NAME GENERATOR


## Introduction
A band name generator is fun tool that combines words or phrases to create unique and memorable names for music bands.
The Band Generator is a beginner project in which **Print** function, **input** function, **variables** and **strings** are used to create the tool.

---

### 1. Write scripts for the following

a. Print welcome message.

```
print("Welcome to the BAND NAME GENERATOR")
```

b. Create input function for users data and assign them variables.

```
food = input("What's the name of your favourite food?\n")
hometown = input("What's the name of your hometown?\n")
```

c. Combine the two input provided by the user to generate name for the band using the print function.

```
print("The name of your band could be: " + food + "-" + hometown)
```
